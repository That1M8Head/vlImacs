vlImacs - Arsalan Kazmi's Emacs config

This config uses Evil.
If you're not a Vimmer, disable it with M-x evil-mode RET.

Get Started
[-----------
| C-x f     - Find a file to open.
| M-x 	    - Run a command.
| C-x 2	    - Open a new window below.
| C-x 3	    - Open a new window to the right.
| C-x t 2   - Open a new tab.
| C-x C-c   - Quit Emacs.
L-----------
